# 제품 요구사항 (PRD)

SafeSpace의 제품 요구사항을 정의하는 문서이다.

---

## Product Overview

| 항목 | 내용 |
|------|------|
| **Product Name** | SafeSpace |
| **Product Type** | Web-based industrial safety monitoring dashboard |
| **Vision** | 실시간 환경 센싱과 작업자 행동 분석을 결합한 밀폐공간 예방적 안전 모니터링 시스템 |

### Problem Statement

밀폐공간 작업은 산소 결핍, 독성 가스 축적, 작업자 이상 인지 지연으로 인해 급속히 치명적 상황으로 발전할 수 있다. 기존 안전 관행은 수동 점검과 사람의 관찰에 의존하므로 빠른 사고 예방에는 한계가 있다.

---

## Target Users

- **현장 안전관리자** — 밀폐공간 작업 감독 및 비상 대응
- **작업구역 관리자** — 구역별 작업 상태 및 환경 모니터링
- **공모전 심사위원 및 발표 청중** — 서비스 가치 평가

---

## MVP Scope

### In Scope

- 실시간 센서 데이터 표시 (O₂, H₂S, CO, VOC, 온도, 습도)
- 위험 등급 산정 및 실시간 대시보드
- 이벤트 로그 및 관리자용 조치 가이드
- 시뮬레이션 모드 (데모 시나리오)
- 영상 기반 간단한 이상행동 탐지

### Out of Scope

- 실제 산업용 인증 하드웨어 연동
- 웨어러블 디바이스 연동, 다중 사업장 통합, 모바일 앱
- 고도화된 예측 AI 모델, 자동 환기 제어

---

## Key Use Cases

1. **실시간 작업구역 상태 확인** — 대시보드에서 센서/작업자 상태 실시간 확인
2. **위험 발생 감지** — 임계치 초과 시 자동 경고 및 상태 전환
3. **작업자 이상행동 감지** — 쓰러짐/무동작 영상 분석 후 이벤트 생성
4. **즉시 조치 확인** — 조치 가이드 패널에서 대응 권고 확인
5. **발표용 시뮬레이션** — 시나리오 버튼으로 시스템 반응 시연

---

## Functional Requirements

| ID | 기능 | 요구사항 |
|----|------|----------|
| FR-01 | Sensor Monitoring | 6종 센서 데이터 표시, 주기적 갱신, 최근 N분 차트 |
| FR-02 | Risk Evaluation | 임계치 판단, 변화량 감지, 복합 조건 위험 레벨 산정 |
| FR-03 | Event Logging | 시간순 이벤트 기록 (유형, 심각도, 시각, 구역, 설명) |
| FR-04 | Worker Monitoring | 작업자 상태 표시, 쓰러짐/무동작 감지 |
| FR-05 | Alerting | WARNING 시각적 경고, CRITICAL 상단 배너 |
| FR-06 | Demo Controls | 시나리오 버튼으로 위험 상태 재현 |

---

## Non-Functional Requirements

| ID | 항목 | 요구사항 |
|----|------|----------|
| NFR-01 | Usability | 첫 화면 3초 이내 목적 이해 |
| NFR-02 | Performance | 1~3초 갱신, 부드러운 인터랙션 |
| NFR-03 | Reliability | 안정적 데모 재현, 데이터 부재 시 UI 유지 |
| NFR-04 | Visual Quality | 다크 테마 산업 관제 UI |

---
